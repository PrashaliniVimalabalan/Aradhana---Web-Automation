# Aradhana Automation Testing Framework

## Overview
This project is a Selenium automation framework developed for the Aradhana Web Application using Python, Selenium, and Pytest.

The framework automates major modules such as:
- Login
- Attendance
- Branch
- User
- Course
- Payment
- Reports
- Settings

## Technologies Used
- Python
- Selenium WebDriver
- Pytest
- PyCharm
- Chrome Browser
